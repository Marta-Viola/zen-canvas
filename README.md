# Zen Canvas - Generative Liquid Art

Un esperimento di arte generativa creato con **HTML5 Canvas** e **JavaScript**.
L'applicazione permette di creare scie fluide con un effetto "mercurio" che cambia colore e spessore in base alla velocità del movimento.

## Features
- **Fluidità dinamica:** lo spessore del tratto reagisce alla velocità del mouse.
- **Ciclo cromatico:** colori HSL fluidi per un effetto ipnotico.
- **Salvataggio:** esporta le tue creazioni in formato PNG.
- **Zero Dependencies:** scritto in Vanilla JavaScript.

## Tech Stack
- HTML5 Canvas API
- CSS3 (Glassmorphism UI)
- JavaScript ES6+
